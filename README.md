# Excel Sales Dashboard Project

## 📊 Tools Used:
- Microsoft Excel (Pivot Tables, Charts, VLOOKUP)

## 📈 Project Description:
This project analyzes sales data using Excel. 
It includes dashboards showing sales by region, category, and monthly trends.

## 🔍 Key Insights:
- West region has highest sales
- Technology category performs best
